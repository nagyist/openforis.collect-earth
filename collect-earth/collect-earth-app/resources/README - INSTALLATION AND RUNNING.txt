INSTALLATION

Simply unzip this file into any folder in your computer.
This self-contained Saiku server will only work in Windows computers!



RUNNING

To run this Saiku instance just double click on the "START SAIKU.bat" file. Once you are finished using Saiku.
Make sure to shutdown the server completely by running the "TERMINATE SAIKU.bat" script (double-click on it)
The browser window should open directly to shoe the Saiku web interface at this address http://127.0.0.1:8181/


USING SAIKU

You can follow the instructions from this tutorial on how to use SAIKU https://openforis.org/tools/collect-earth/tutorials/saiku/.
There is also more information in this PDF https://openforis.org/wp-content/uploads/2021/03/Open_Foris_Saiku_Manual_November2018.pdf
